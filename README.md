# 📦 Tastade PyPI Mirror

Персональное зеркало pip-пакетов для **Termux**.

---

## Быстрая установка

```bash
pip install piphub-repo
piphub-repo use tastade
```

## Использование

```bash
piphub-repo install requests
piphub-repo install requests rich tqdm
piphub-repo ping tastade
```
